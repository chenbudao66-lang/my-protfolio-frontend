# Channing Winchester - 全栈作品集网站

这是一个使用 React 构建的现代化全栈个人作品集网站，展示了项目作品、技术博客和个人信息。

## 🚀 功能特性

- **响应式设计** - 完美适配桌面和移动设备
- **多页面导航** - 使用 React Router 实现单页应用
- **用户认证** - 完整的登录/登出功能
- **管理后台** - 受保护的管理界面
- **API 集成** - 与后端 RESTful API 通信
- **现代化 UI** - 使用统一的色彩方案和优雅的设计

## 🎨 设计特色

- **色彩方案**: #4C2111 (深红棕), #098897 (青蓝), #D6C8AD (米白)
- **艺术风格**: 融合古典美学与现代设计
- **用户体验**: 注重细节的交互设计
